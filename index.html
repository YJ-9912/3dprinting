모델링 파일 변환

3D 프린터는 표준 파일 형식으로 STL (STereoLithography)을 사용합니다.
G-code는 사람이 컴퓨터화된 공작기계에 어떤 것을 어떻게 만들지를 명령하는데 사용하는 언어이며
3D 프린터의 노즐 등이 이동하는 경로 혹은 속도 등을 표현할 수 있습니다.

프린터기 사용 전 프린터 사용법과 출력 상세 설정법을 반드시 숙지 부탁드립니다.
사용 시 문제가 있을 경우, 제플 관리자에게 연락 바랍니다.

  • C4D에서 STL파일로 저장하는 법
  • G-Code 파일로 변형하는 법

최종 모델링 후 1,2,3 순으로 클릭한 후 저장합니다.

1. 제플실 컴퓨터(비밀번호는 화이트보드 상단)에서 Cubicreater를 켭니다.

2. 1,2번 프린터 사용 시
Cubicreater 3.6.8의 프린터 아이콘(출력옵션)을 클릭합니다.

3. 상세 설정에서 왼쪽 하단의 G-code 다운로드로 저장합니다.

4. 3,4번 프린터 사용 시 Cubicreater 4.2.4의 출력준비를 클릭합니다.

5. 상단의 G-code 저장을 누릅니다.
